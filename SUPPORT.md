# Support

AudioPilot is maintained as a small open-source project.

## Getting Help

Before opening an issue:

1. Check that you are running macOS 13 Ventura or newer.
2. Confirm that notifications are enabled for AudioPilot in System Settings.
3. Quit and relaunch the app.
4. Try changing the default output device from Control Center or System Settings.

## Open An Issue

Use GitHub Issues for:

- Reproducible bugs.
- macOS compatibility reports.
- Documentation fixes.
- Small feature requests that fit the privacy-first scope.

Please include your macOS version, AudioPilot version, hardware context, and the output devices involved.

## Security

Do not open a public issue for sensitive security reports. See [SECURITY.md](SECURITY.md).
